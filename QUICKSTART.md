# Quick Start Guide

Welcome! This is your crash course in Linux fundamentals for web hosting work at Cloud86.io.

## âš¡ Get Started in 3 Minutes

### 1. Open WSL2 Terminal

Press `Windows Key` + type "Ubuntu" or "WSL" and hit Enter.

### 2. Clone This Course

```bash
cd ~
git clone https://github.com/cloud86io/linux-fundamentals-wsl.git
cd linux-fundamentals-wsl
```

### 3. Start Module 1

```bash
cd modules/01-navigation-basics
cat lesson.md
```

Read through the lesson, then:

```bash
cat exercises.md
```

Work through the exercises. When done:

```bash
bash validate.sh
```
